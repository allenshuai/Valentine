# Valentine

A small mobile page — open the link, tap **how**, and watch the bouquet open.

## Host on GitHub Pages

1. Create a new repo (e.g. `valentine`), push this folder.
2. **Settings → Pages** → Source: **Deploy from a branch**.
3. Branch: `main`, folder: **/ (root)** → Save.
4. Your link will be: `https://YOUR_USERNAME.github.io/valentine/`

No build step — just `index.html`.
